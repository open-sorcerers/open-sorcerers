* Brian Lonsdorf - [Mostly Adequate Guide to FP] / [`daggy`][daggy]
* Kyle Simpson - [FP Light] / [`fpo`][fpo]
* Reginald Braithwaite - [JavaScript Allonge]
* Xuanyi Chew - [Underhanded JavaScript]
* Eric Elliott - [Composing Software]
* Tom Harding - [Fantas, Eel, and Specification]
* Quil - [A Monad In Practicality]
* Scott Sauyet - [Functional Programming In JavaScript] / [`ramda`][ramda]
