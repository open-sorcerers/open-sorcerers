## [Composition](/glossary/composition)

An operation which takes two or more functions and returns a unary function.

```js
const f = x =>  x
const g = x => y
const h = x => f(g(x))
// or, es6 style:
import {pipe} from 'ramda'
const h = pipe(f, g)
```
