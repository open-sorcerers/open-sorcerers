## [Ternary](/glossary/ternary)

A function which takes three parameters, or is said to have an [arity](/glossary/arity) of 3.

```js
const ternary = (a, b, c) => a + b + c
const ternaryMC = a => b => c => a + b + c 
const ternaryC = curry((a, b, c) => a + b + c)
```
