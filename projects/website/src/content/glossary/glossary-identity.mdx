# [Identity](/glossary/identity)

The identity function returns what it is given. Sometimes referenced as the `I` combinator.

```js
const I = x => x
// import { identity as I } from 'ramda'
'so\n\n\n\n\nso\n\n\ncool'.split('\n').filter(I)
```
