## Purity

A pure function is [deterministic](/glossary/deterministic), has no side-effects and is [referentially transparent](/glossary/refentially-transparent).

```js
const add = curry((a, b) => a + b)
const addTwo = add(2)
addTwo(100) / add(200)(111)
(2+)(100) / (200+)(111)
102 / 311
```
